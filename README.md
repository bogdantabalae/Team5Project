# Game Store Platform

## Overview

Game Store Platform is a **college group project** developed to demonstrate full-stack web development skills using modern technologies. The platform is an online marketplace where users can browse digital game products, create accounts, log in securely, and purchase game activation codes.

The system is built using a three-tier architecture with separate frontend, backend, and database services deployed in the cloud.

---

## Project Purpose

This project was created as part of a **school assignment** to showcase:

- Frontend web development
- Backend API development
- Database integration
- User authentication
- Cloud deployment
- Team collaboration

---

## Features

- User registration and login
- Browse available games
- View product details
- Purchase digital game keys
- Responsive user interface
- Secure REST API integration
- Cloud-hosted database
